Elementos Basicos

Un sistema estructurado que permite etiquetar archivos de texto para conseguir diversos efectos en paginas web.

Un lenguaje de programacion que a través de stylesheets (hojas de estilo) permite controlar el aspecto de documentos HTML y XHTML.

Un lenguaje de programacion que permite crear y agregar contenido dinamico a archivos HTML y XHTML.

HTML

Para darle estructura a tu pagina

HTML es el lenguaje con el que se define el contenido de las páginas web. Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web, como imágenes, listas, vídeos, etc.

Ejemplo

<!DOCTYPE html>
<html>
 <head>
  <title>Titulo</title>
 </head>
 <body>
  <div>
   <h1>Primer Encabezado</h1>
   <p>Primer parrafo.</p>
  </div>
 </body>
<html>

Explicacion

<!DOCTYPE html> Este elemento define el tipo de lenguaje del documento, en este caso html5.
<html> Este elemento define la raiz del documento.
<head> Este elemento define el contenedor de metadatos, es decir titulos, enlaces, entre otros.
<title> Este elemento define el titulo de la pagina web que se muestra en la pestaña del navegador.
<body> Este elemento define el cuerpo del documento, es decir el contenedor del resto de elementos, encabezados, parrafos, imagenes, hipervinculos, entre otros.
<div> Este elemento define un contenedor, que es un subespacio dentro del cuerpo del documento
<h1> Este elemento define un encabezado, existen hasta 6 tamaños
<p> Este elemento define un parrafo

Es la quinta versión de HTML que presenta las siguientes características:

  • Compatibilidad con elementos graficos vectoriales como SVG o Canvas
  • Compatibilidad con dispositivos moviles
  • Permite integrar elementos de audio y video en paginas web
  • Contenedores propios como <header>, <nav>, <section>, <article>, <footer>
