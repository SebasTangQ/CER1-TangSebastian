Elementos Basicos para una Pagina Web

Elementos Basicos

Un sistema estructurado que permite etiquetar archivos de texto para conseguir diversos efectos en paginas web.

Un lenguaje de programacion que a través de stylesheets (hojas de estilo) permite controlar el aspecto de documentos HTML y XHTML.

Un lenguaje de programacion que permite crear y agregar contenido dinamico a archivos HTML y XHTML.

HTML

Para darle estructura a tu pagina web

Informacion Inicial

Es un lenguaje con el que se define la estructura jerarquica y el contenido de un archivo de texto, el cual puede ser interpretado por un navegador como página web.

Ejemplo

<!DOCTYPE html>
<html>
 <head>
  <title>Titulo</title>
 </head>
 <body>
  <div>
   <h1>Primer Encabezado</h1>
   <p>Primer parrafo.</p>
  </div>
 </body>
<html>

Explicacion

<!DOCTYPE html> Este elemento define el tipo de lenguaje del documento, en este caso html5.
<html> Este elemento define la raiz del documento.
<head> Este elemento define el contenedor de metadatos, es decir titulos, enlaces, entre otros.
<title> Este elemento define el titulo de la pagina web que se muestra en la pestaña del navegador.
<body> Este elemento define el cuerpo del documento, es decir el contenedor del resto de elementos, encabezados, parrafos, imagenes, hipervinculos, entre otros.
<div> Este elemento define un contenedor, que es un subespacio dentro del cuerpo del documento
<h1> Este elemento define un encabezado, existen hasta 6 tamaños
<p> Este elemento define un parrafo

Son formas de clasificar los distintos elementos dentro del cuerpo de un documento donde:

  • Etiqueta o Nametag: Hace referencia al constructor del elemento <body>, <div>, <h1>, <p>
  • Identificador o ID: Hace referencia a un identificador unico de un elemento, se declara dentro del constructor <div id="informacion">
  • Clase o Class: Hace referencia a una clase de la que uno o mas elementos son pertenecientes, se declara dentro del constructor <p class="texto">

Es la quinta versión de HTML que presenta las siguientes características:

  • Compatibilidad con elementos graficos vectoriales como SVG o Canvas
  • Compatibilidad con dispositivos moviles
  • Permite integrar elementos de audio y video en paginas web
  • Contenedores propios como <header>, <nav>, <section>, <article>, <footer>

CSS

Para agregar colores, fuentes e imagenes a tu pagina web

Informacion Inicial

Es un lenguaje de programacion que permite darle estilo, forma, color y posicion a los diferentes elementos descritos en un documento HTML

Ejemplo

body {
background-color: lightblue;
}
#informacion {
height: 30px;
width: auto;
}
.texto {
color: grey;
text-align: center;
}

Explicacion

body hace referencia a un elemento tipo tag que sera modificado, en este caso el cuerpo del documento.
background-color indica el color de fondo del elemento, su valor puede ser escrito con palabras, codigo hex o rgb, entre otros
#informacion hace referencia al elemento que tenga como identificador "informacion".
height indica la altura del elemento, dicho valor se encuentra expresado en pixeles
width indica la anchura del elemento, el valor se define como automatico, por lo general el 100% del ancho de la ventana
.texto hace referencia a todos los elementos que contengan como clase "texto".
color indica el color del elemento, acepta el mismo formato que el atributo background-color
text-align indica la alineacion del texto del elemento, en este caso centrado, aunque podria ser alineado a la izquierda (left) o derecha (right)

Para implementar un archivo tipo .css en un documento HTML se enlaza a través de la tag <link> en el contenedor <head>:

Ejemplo

<!DOCTYPE html>
<html>
 <head>
  <title>Titulo</title>
  <link rel="stylesheet" href="css/estilo.css"></link>
 </head>
 <body>
  <div>
   <h1 class="texto">Primer Encabezado</h1>
   <p id="informacion">Primer parrafo.</p>
  </div>
 </body>
<html>

Explicacion

<link> Este elemento define una relacion entre el documento HTML y un recurso externo
rel indica el tipo de relacion entre el documento HTML y el recurso externo, este caso stylesheet es para indicar una relacion por hoja de estilo
href indica la ruta de ubicacion del recurso externo, en este caso la ruta es css/estilo.css

JavaScript

Para agregar elementos dinamicos a tu pagina web

Informacion Inicial

Es un lenguaje de programacion interpretado que permite crear funciones, variables, procesos y elementos dinamicos que pueden ser vinculados a un archivo html

Para implementar un archivo tipo .js en un documento HTML se enlaza a través de la tag <script> al final del contenedor <body>:

Ejemplo

<!DOCTYPE html>
<html>
 <head>
  <title>Titulo</title>
 </head>
 <body>
  <div>
   <h1 class="texto">Primer Encabezado</h1>
   <p id="informacion">Primer parrafo.</p>
   <img id="logotipo" src="logo.png">
  </div>
 </body>
<html>
